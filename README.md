项目描述：
视频生成器是一款基于最新人工智能生成内容（AIGC）技术的创新工具。该工具通过调用GPT系列及头部模型，根据用户输入的主题调用维基百科等API联网查询最新相关信息，作为提示词的上下文语境。系统将结合视频时长及词义随机性要求输出合适的脚本内容。适用于内容创作者、企业宣传等多种应用场景。

项目概述：
基于LangChain和维基百科API的短视频文案生成工具，能够根据用户输入的主题、时长和创意度参数，自动生成吸引人的短视频标题和结构化脚本内容。

技术架构：
- 前端框架：Streamlit - 提供参数配置和结果展示界面
- AI框架：LangChain - 构建多步骤生成链
- 外部知识：WikipediaAPIWrapper - 获取主题相关信息
- 提示工程：ChatPromptTemplate - 结构化提示词管理
- 大语言模型：GPT-4-turbo - 内容生成引擎

核心功能：
- 智能标题生成：基于主题生成吸引流量的短视频标题
- 结构化脚本：按照【开头】-【中间】-【结尾】的格式生成完整脚本
- 时长适配：根据用户指定的视频时长调整内容长度
- 创意度控制：通过temperature参数控制内容的创新程度
- 知识增强：结合维基百科信息提升内容准确性
- 风格定制：针对95后上班族群体优化内容风格

技术亮点：
- 采用多步骤生成链，先生成标题再生成脚本
- 集成外部知识源，提升内容质量和准确性
- 可调节的创意度参数，满足不同创作需求
- 结构化的输出格式，便于后续视频制作
- 异常处理机制，确保系统稳定性

应用场景：
- 短视频内容创作
- 社交媒体营销
- 教育培训视频
- 产品宣传片
- 个人IP打造

项目价值：
展示了AIGC在内容创作领域的强大能力，通过技术手段降低了短视频创作的门槛，体现了对内容创作流程的深度理解和产品化思维，为内容创作者提供了高效的生产工具。
